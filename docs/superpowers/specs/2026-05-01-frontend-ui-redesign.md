# 脚本管理系统前端UI重设计

## 设计概述

本次重设计采用**渐变玻璃风格**，以紫蓝渐变为背景基调，配合边框玻璃卡片和圆润填充交互元素，打造现代化、视觉层次丰富的技术工具界面。

### 设计选择摘要

| 选择项 | 决定 |
|--------|------|
| 整体风格 | 渐变玻璃风格 |
| 渐变配色 | 紫蓝渐变 (#667eea → #764ba2) |
| 页面布局 | 紧凑侧边栏布局 (约55px) |
| 卡片样式 | 边框玻璃风格 (透明背景+发光边框) |
| 交互元素 | 圆润填充风格 (圆角胶囊形状) |

---

## 配色系统

### 主渐变背景

```css
/* 全局渐变背景 */
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```

### 语义色彩

| 用途 | 颜色值 | 使用场景 |
|------|--------|----------|
| Primary | `#667eea` / `#764ba2` | 主要按钮、Logo、高亮元素 |
| Success | `#67c23a` | 成功状态、完成提示 |
| Warning | `#e6a23c` | 警告提示、待处理状态 |
| Danger | `#f56c6c` | 错误提示、删除操作 |
| Info | `#909399` | 普通信息、次要提示 |

### 玻璃透明度层级

| 层级 | 透明度 | 使用场景 |
|------|--------|----------|
| 高亮背景 | `rgba(255,255,255,0.25)` | 选中状态、主要按钮 |
| 活动背景 | `rgba(255,255,255,0.15)` | 悬停状态、活动tab |
| 基础背景 | `rgba(255,255,255,0.10)` | 输入框、次要按钮 |
| 低亮背景 | `rgba(255,255,255,0.05)` | 分隔线、阴影区域 |

### 边框颜色

| 类型 | 颜色值 | 使用场景 |
|------|--------|----------|
| 主边框 | `rgba(255,255,255,0.35)` | 卡片外边框、主要输入框 |
| 次边框 | `rgba(255,255,255,0.20)` | 内部分隔、次要元素 |
| 低亮边框 | `rgba(255,255,255,0.10)` | 状态栏、底部区域 |

### 侧边栏 (55px)

- **位置**: 左侧固定
- **样式**: `rgba(255,255,255,0.08)` 背景 + `rgba(255,255,255,0.15)` 边框
- **圆角**: `12px`
- **内容**: 图标式导航按钮 (32x32px)
- **Logo**: 渐变填充的小方块 (32x32px)

### 顶部标题栏

- **高度**: 50px
- **样式**: 透明背景 + 白色边框 (rgba(255,255,255,0.15))
- **圆角**: `12px`
- **内容**: 页面标题 + 操作按钮组 + 用户头像

### 主内容区

- **布局**: CSS Grid / Flexbox
- **卡片间距**: `12px`
- **响应式**: 自适应列数 (3列 → 2列 → 1列)

### 底部状态栏

- **高度**: 40px
- **样式**: 透明背景 + 低亮边框
- **内容**: 统计信息 + 系统状态

## 组件样式规范

### 卡片组件

```css
.card {
  background: transparent;
  border: 1px solid rgba(255, 255, 255, 0.2);
  border-radius: 12px;
  padding: 15px;
  transition: all 0.3s ease;
}

.card:hover {
  border-color: rgba(255, 255, 255, 0.35);
  background: rgba(255, 255, 255, 0.05);
}
```

### 按钮样式

```css
/* 主要按钮 - 圆润填充 */
.btn-primary {
  background: rgba(255, 255, 255, 0.15);
  border-radius: 20px;
  padding: 8px 16px;
  color: #fff;
  font-weight: 500;
  transition: all 0.2s ease;
}

.btn-primary:hover {
  background: rgba(255, 255, 255, 0.25);
}

/* 次要按钮 */
.btn-secondary {
  background: rgba(255, 255, 255, 0.08);
  border-radius: 20px;
  padding: 8px 16px;
  color: rgba(255, 255, 255, 0.8);
}

/* 小型操作按钮 */
.btn-sm {
  border-radius: 12px;
  padding: 4px 10px;
  font-size: 10px;
}
```

### 输入框样式

```css
.input {
  background: rgba(255, 255, 255, 0.15);
  border-radius: 20px;
  padding: 10px 14px;
  color: rgba(255, 255, 255, 0.9);
  border: none;
}

.input:focus {
  background: rgba(255, 255, 255, 0.2);
  outline: none;
}

.input::placeholder {
  color: rgba(255, 255, 255, 0.5);
}
```

### 表格样式

- **表头**: 透明背景 + 底部边框
- **行**: 交替透明度 (hover时增加背景)
- **边框**: 仅分隔线，无边框

### 代码编辑器

- **背景**: 深色半透明 `rgba(30, 30, 50, 0.85)`
- **边框**: 渐变边框 `rgba(102, 126, 234, 0.3)`
- **语法高亮**: 保持现有方案，适配深色主题

## 深色/浅色主题

本次设计以深色渐变背景为主，同时提供浅色模式切换：

### 浅色模式

```css
/* 浅色模式 - 保持渐变但降低对比度 */
background: linear-gradient(135deg, #f0f4ff 0%, #e8e0f0 100%);

/* 卡片改为白色半透明 */
.card {
  background: rgba(255, 255, 255, 0.7);
  border-color: rgba(102, 126, 234, 0.2);
}

/* 文字改为深色 */
color: #333;
```

